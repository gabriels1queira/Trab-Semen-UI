Existem duas maneiras de incluir o Semantic UI no seu site:

  • Importando os arquivos

    1. Baixando pelo Site Oficial ou clonando o Repositório do Github;
    2. Inserir os arquivos no diretório do site;
    3. Usar a tag <link> para acessar o framework baixado.

  • Importando o CDN

    1. Copiar os CDN`s necessários para o seu site em: CDNJS;
    2. Usar a tag <link> para acessar o CDN.
